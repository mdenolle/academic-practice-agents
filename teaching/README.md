# teaching/

The teaching agent supports a faculty member in designing and producing instructional artifacts — lecture notes, slides, and Jupyter notebooks — grounded in their own course materials and pedagogical philosophy. It acts as a faculty collaborator, not a content generator.

## Specs in this folder

- [seismo-course-agent.md](seismo-course-agent.md) — generates JupyterBook notes, slides, and in-class notebooks for an undergraduate/graduate seismology course
